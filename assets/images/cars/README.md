# Imágenes de Vehículos

Coloca aquí las imágenes de los vehículos para el carrusel de la pantalla de bienvenida.

## Archivos requeridos:

- `sedan.jpg` - Imagen de un sedán
- `suv.jpg` - Imagen de un SUV
- `van.jpg` - Imagen de una van
- `luxury.jpg` - Imagen de un vehículo de lujo

## Formato recomendado:

- Formato: JPG o PNG
- Resolución: 1920x1080 o superior
- Tamaño: Optimizado para web (máximo 500KB por imagen)

Las imágenes se mostrarán en un carrusel en la pantalla de bienvenida. Si una imagen no existe, se mostrará un placeholder con el nombre del vehículo.
